# GreenLeaf Store - React Plant E-commerce Application\n\n## Project Overview\nGreenLeaf Store is a modern React-based e-commerce web application for selling houseplants, featuring Redux state management and shopping cart functionality.\n\n## GitHub Repository\n**Public URL:** https://github.com/101Krishna/plant-store-react\n\n## Features\n\n### Landing Page\n- Welcome hero section with \"GreenLeaf Store\" branding\n- Company description and features section\n- \"Get Started\" button linking to product listing\n\n### Product Listing Page\n- **6 Unique Houseplants with prices:**\n  1. Monstera Deliciosa ($45)\n  2. Pothos Golden ($25)\n  3. Snake Plant ($35)\n  4. Fiddle Leaf Fig ($60)\n  5. Peace Lily ($30)\n  6. Rubber Plant ($50)\n\n- **3+ Categories:** Indoor, Outdoor, Flowering\n- Add to Cart functionality with button state management\n\n### Header Navigation\n- Displays on all pages\n- Shopping cart icon with item count\n- Navigation links to all pages\n\n### Shopping Cart Page\n- Total items and total price display\n- Increment/Decrement quantity buttons\n- Delete item functionality\n- \"Coming Soon\" checkout button\n- Continue Shopping link\n\n## Redux Store\n- **cartSlice.js:** Cart reducer with add, remove, increment, decrement actions\n- **store.js:** Redux store configuration with cart reducer\n\n## Project Structure\n```\nsrc/\n├── components/\n│   ├── Header.jsx\n│   └── Header.css\n├── pages/\n│   ├── LandingPage.jsx\n│   ├── ProductListing.jsx\n│   └── ShoppingCart.jsx\n├── store/\n│   ├── cartSlice.js\n│   └── store.js\n└── App.js\n```\n\n## Technologies\n- React 18.2.0\n- React Router 6.0.0\n- Redux 4.2.0 with Redux Toolkit 1.9.0\n- React-Redux 8.0.0\n\n## Installation\n```bash\ngit clone https://github.com/101Krishna/plant-store-react.git\ncd plant-store-react\nnpm install\nnpm start\n```
